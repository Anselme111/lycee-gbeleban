## 🖼️ Comment ajouter vos photos

1. Prenez vos photos JPG (ou PNG, WebP)
2. Renommez-les exactement comme indiqué ci-dessus
3. Copiez-les dans le dossier `images/`
4. Les placeholders SVG bleus seront automatiquement remplacés

**Taille recommandée :** 1200×800 px minimum pour les grandes photos, 800×600 pour les autres.

## 🚀 Hébergement GitHub Pages

1. Créer un dépôt GitHub (ex: `lpg-gbeleban`)
2. Uploader tous les fichiers
3. Settings → Pages → Source: main branch
4. Votre site sera disponible sur : `https://votre-nom.github.io/lpg-gbeleban/`

## 🎨 Personnalisation

Les couleurs et polices sont dans `css/main.css` — section `:root` :
- `--B` : Bleu principal `#1447A6`
- `--O` : Orange `#F07520`
- Police titres : Playfair Display
- Police corps : Outfit
